# 🧩 Interactive Sudoku Game

A dynamic and interactive Sudoku game built from scratch using pure HTML, CSS, and Vanilla JavaScript. The project aims to provide a clean, modern, and engaging user experience while implementing classic Sudoku mechanics.

Currently in active development (Work in Progress).

## ✨ Key Features

- **Dynamic Board Generation (Shuffle Logic):** The game doesn't just use a static board. On every page refresh, the rows, columns, and digits are shuffled randomly based on a valid Sudoku template, ensuring a unique puzzle every time you play.
- **Interactive UI & 3D Buttons:** Modern, responsive design with 3D-styled number buttons and hover effects for an intuitive user experience.
- **Visual Feedback:** - Instant green highlight for correctly placed numbers.
  - Animated red pop-up for mistakes.
- **Mistake Tracking:** Classic rules apply — you are allowed a maximum of **5 mistakes**. Make a 6th mistake, and it's Game Over!
- **Time Tracking:** A built-in timer tracks how long it takes you to solve the puzzle.
- **Win/Loss States:** The game automatically detects when the board is successfully completed or when the maximum error limit is reached, showing appropriate end-game messages.

## 🛠️ Technologies Used

- **HTML5:** Structuring the game board and UI.
- **CSS3:** Styling, grid layouts, 3D button effects, and smooth animations.
- **Vanilla JavaScript:** Game logic, DOM manipulation, board validation, and dynamic generation. No external libraries or frameworks were used.

## 🚀 How to Run the Game

Since this is a plain front-end application, running it is incredibly simple and requires no installations or servers:

1. Download the project files to your computer (either by cloning the repository or downloading the ZIP file).
2. Open the folder where you saved the files.
3. Double-click on the `index.html` file to open it directly in any modern web browser.
4. Have fun playing!
